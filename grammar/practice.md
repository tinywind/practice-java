#1. 두 정수 더하기(A+B)
* 입력값 A와 B의 합을 구하는 함수를 기술하시오.
* 함수의 원형은 다음과 같다.
```
 int sum(int A, int B)
```
* A와 B의 범위는 Integer.MIN_VALUE 에서 Integer.MAX_VALUE 이다.
* 결과값은 A와 B의 합이다.
* 만약 결과값이 Integer 의 표현 범위를 벗어나면, -1을 돌려준다.
